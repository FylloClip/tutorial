# FylloClip Tutorial

Build your own low-cost foliar sensor for monitoring plant water status using LoRaWAN.

This tutorial guides you step by step from hardware assembly to cloud-based data transmission.

---

## Start here

Full Tutorial:

---

## What is FylloClip?

FylloClip is a leaf-mounted capacitive sensor designed to provide qualitative information on the water status of plants.

It allows:
- continuous monitoring of plant water status
- remote data transmission, e.g. via LoRaWAN
- low-cost and scalable deployment

---

## What you will learn

This tutorial covers:

- Ordering PCBs and electronic components  
- Soldering and assembling the device  
- Uploading and configuring Arduino code  
- Connecting the device to a LoRaWAN network  
- Accessing and interpreting sensor data
